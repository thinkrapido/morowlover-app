# MorowLover Desktop App

based on electron and angular

** Currently under Development **

Linux:
```
> webpack
> cp src/electron/electron.js dist
> electron .
```

DOS:
```
> webpack
> copy src\electron\electron.js dist
> electron .
```

### Prerequisites

obviously ```yarn global add electron webpack``` or ```npm install --global electron webpack```
    
then ```yarn install``` or ```npm install```
